# CodeStudio

Desktop GUI for Claude Code.

## Features

- Multi-tab Claude Code sessions
- Project file browser with CLAUDE.md support
- Custom AI agents with MCP integration
- Session history and analytics
- Cross-platform (Windows, macOS, Linux)

## Quick Start

```bash
# Install dependencies
bun install

# Run development
bun run tauri dev

# Build for production
bun run tauri build
```

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Rust, Tauri
- **State**: Zustand

## Requirements

- [Bun](https://bun.sh) or Node.js
- [Rust](https://rustup.rs)
- [Claude Code CLI](https://claude.ai/code)
